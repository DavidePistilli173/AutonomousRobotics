GUIDA ALL'INSTALLAZIONE
********************************************* Task *********************************************

Abbiamo creato un file launch dove abbiamo inserito il nome del topic (/hw1_target_objects) che utilizzeremo per rilevare le pose degli oggetti posti sull'area di lavoro.
Abbiamo impostato un valore di Simulation, che indicherà se l'algoritmo verrà eseguito in un ambiente Reale(0) o Simulazione(1), il valore di default è Reale(0).

1) catkin build

2) roslaunch hw2 hw2.launch simulation:= (0/1)

3) Indicare da tastiera in quale DockingStation si vuol rilasciare l'oggetto,
   - DockingStation1 = 1;
   - DockingStation2 = 2;
